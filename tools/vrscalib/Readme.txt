VRS Calibration

Requirements
- numpy - required by VRSCalib.py and optimize.py.
- scipy - only scipy.optimize.optimize is required. The module is
  included in this directory. (Modification: unneeded import statement
  commented out.)

Usage
- Startup: Right click VRSCalib.x3d -> send to H3D
- Follow on-screen instructions, press H for help.
- Move generated device.txt/viewpoint.txt to appropriate H3D settings folder (rename as necessary).

Tips
- Use a piece of glass instead of a mirror to be able to see the real
  stylus
